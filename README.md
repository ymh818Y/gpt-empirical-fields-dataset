# GPT對實證驅動型領域學術論文的影響研究——數據集

## 數據集簡介

本數據集為論文《GPT對實證驅動型領域學術論文的影響研究》的支撐數據。

研究將2022年11月GPT-3.5的發布視為準自然實驗的外生衝擊，基於OpenAlex全球開放學術數據庫，選取金融科技、公共安全、醫療大數據、城市交通、生態環境5個典型實證領域2019—2024年論文元數據為樣本，構建連續時間斷點回歸（RDiT）模型，分析GPT對實證驅動型領域學術論文發表的淨因果衝擊效應。

## 數據來源

- **數據源**：OpenAlex全球開放學術數據庫（https://openalex.org/）
- **採集時間**：2025年1月
- **時間範圍**：2019年1月1日 — 2024年12月31日
- **樣本規模**：30,000篇論文（5個領域各6,000篇）
